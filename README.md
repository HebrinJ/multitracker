# Мультитрекер (React 19 + Next.js 16)

Веб-приложение для отслеживания прочитанных книг и просмотренных фильмов.

## Установка

1. Скопируйте файлы в папку вашего проекта (кроме package.json - у вас уже есть правильный)
2. Установите зависимости: `npm install` (уже установлены)
3. Создайте базу данных: `npx prisma db push`
4. Запустите: `npm run dev`

## Копирование файлов

Скопируйте следующие папки из архива:

```
src/app/        → замените файлы (кроме layout.tsx и globals.css)
src/components/ → скопируйте папку целиком
src/lib/        → скопируйте папку целиком
src/types/      → скопируйте папку целиком
src/providers/  → скопируйте папку целиком
src/hooks/      → скопируйте папку целиком
prisma/         → скопируйте папку целиком
```

## Структура

```
src/
├── app/              # Страницы
│   ├── books/        # Каталог книг
│   ├── movies/       # Каталог фильмов
│   ├── auth/         # Авторизация
│   └── profile/      # Профиль
├── components/
│   ├── app/          # Компоненты приложения
│   └── ui/           # UI компоненты
├── lib/              # Библиотеки
├── types/            # Типы TypeScript
└── providers/        # React провайдеры
```

## Технологии

- React 19
- Prisma + SQLite
